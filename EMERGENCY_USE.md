# 🚨 긴급 재게시 버전

## 이 버전의 특징

```python
FORCE_UPDATE = True  # ← 하드코딩됨!
```

- ✅ 무조건 모든 기사 재번역
- ✅ Secret 설정 불필요
- ⚠️ 한 번만 사용 후 원래 버전으로 복구!

## 🚀 사용 방법

### 1. 긴급 재게시

1. 이 `translate_and_post.py` 파일로 **교체**
2. GitHub 커밋
3. Actions → Run workflow
4. ✅ 10개 기사 재번역 시작!

### 2. 완료 후 원복

**중요:** 한 번 실행 후 반드시 원래 코드로 되돌리세요!

```python
# 이렇게 변경:
FORCE_UPDATE = False
```

또는 `drone-final` 폴더의 원본 파일로 교체

## ⚠️ 주의사항

이 버전을 그대로 두면:
- 매일 자동 실행 시 계속 중복 게시
- 같은 기사가 매일 올라감

**사용 시나리오:**
1. 긴급 재게시 필요 → 이 버전 사용
2. 한 번 실행 완료
3. 즉시 원본으로 교체 ← 필수!
